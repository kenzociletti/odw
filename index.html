Feuille CSS de base

Ces propriétés permettent de fixer le footer (pied de page) du site en bas quel que soit le contenu de la page. 

          
* {
  box-sizing: border-box;
}

html, body {
  height: 100%;
}

body {
  margin: 0;
  display: flex;
  flex-direction: column;
}

main {
  padding: 40px;
  flex-grow: 1;
}

section {
  max-width: 1200px;
  margin: auto;
}

footer {
  text-align: center;
}
